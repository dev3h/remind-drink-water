## Generate venv
```bash
python -m venv venv
```

## Activate venv
```bash
./venv/Scripts/activate
```

## Install dependencies
```bash
pip install -r requirements.txt
```

## Run project
```bash
python index.py
```

## Build dự án
Chạy lệnh sau để build dự án ra file exe:
```
pyinstaller index.spec
```
